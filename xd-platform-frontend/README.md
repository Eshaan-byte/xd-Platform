# 🕹️ XD Platform — Frontend

This project implements the XD Platform landing page based on Figma designs, featuring Featured Games, Trending, XDGSS Specials, Sales, Community Picks, and more.

---

## 🚀 Tech Stack

| Technology | Purpose |
|-----------|---------|
| **React + Vite** | Fast development environment and modern build tool |
| **TypeScript** | Strongly typed components |
| **Tailwind CSS** | Utility-first styling matching the Figma design |
| **PostCSS + Autoprefixer** | CSS processing |
| **ESLint** | Code linting |

---

## 🛠️ Local Development Setup

### Clone the repository

```bash
git clone https://github.com/<your-username>/xd-platform-frontend.git
cd xd-platform-frontend
```

### Install dependencies

```bash
npm install
```

### Start the dev server

```bash
npm run dev
```
